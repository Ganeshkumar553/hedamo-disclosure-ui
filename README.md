This project was completed as part of an **internship assignment** focused on
designing a frontend interface for a **producer disclosure platform**.

Hedamo is **not a verification or certification system**.
It displays **producer-declared information** transparently and clearly.

---

## Objective

- Translate the concept of **disclosure over verification** into UI design
- Avoid approval or validation language
- Emphasize clarity, responsibility, and transparency

---

## Key Features

- Product List view with required disclosure attributes
- Product Detail view with:
  - Producer-declared summary
  - Always-visible disclaimer
  - Version history
- Neutral workflow status indicators
- Search and empty-state feedback
- Clean, professional UX

---

## Tech Stack

- Frontend framework: React
- Plain CSS
- Static mock data

---

## Notes

- Frontend-only implementation
- No backend or verification logic
- UI designed to support future system expansion
